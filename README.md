# ChatCord App

ChatCord is a real-time chat application built using Node.js, Express, and Socket.io on the backend, with Vanilla JavaScript powering the frontend. It offers a custom UI for a seamless chatting experience.

## Usage

1. **Installation**: First, ensure you have Node.js installed on your system. Then, navigate to the project directory in your terminal and run:

   ```
   npm install
   ```

2. **Run the Server**: Start the development server by running:

   ```
   npm run dev
   ```

3. **Access the App**: Open your web browser and visit [localhost:3000](http://localhost:3000) to access the ChatCord application.

## Notes

- The `_html_css` folder included in this repository serves as a starter template to accompany the tutorial available at [YouTube Tutorial](https://www.youtube.com/watch?v=jD7FnbI76Hg&t=1339s). It is not part of the actual ChatCord application.

This README provides a brief overview of the ChatCord app, along with instructions for setting it up and accessing it. For further details and documentation, please refer to the repository.
